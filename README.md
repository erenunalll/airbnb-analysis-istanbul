## Project Description
The aim of this project is to analyze the Airbnb listings in Istanbul by finding answers to the questions: What are the factors that influence the Airbnb prices and how do these factors influence the prices in Istanbul? By applying statistical methods such as hypothesis test and Analysis of Variance (ANOVA), this project answers these questions and gives statistical proofs about the features that has a significant affect on the Airbnb prices in Istanbul.

---

## Data
Source: InsideAirbnb
- 22,865 observations
- 15 variables
- Included variables:
  - id
  - host_id
  - neighbourhood
  - latitude
  - longitude
  - room_type
  - price
  - minimum_nights
  - number_of_reviews
  - last_review
  - reviews_per_month
  - calculated_host_listings_count
  - availability_365
  - number_of_reviews_ltm
  - side (Europe/Anatolia)
